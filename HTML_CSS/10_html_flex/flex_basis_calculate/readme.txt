1. Встановити
    .item1 { flex-grow: 1; }
    .item2  { flex-grow: 2;  }

При цьому за умовчанням flex-basis -> auto 
і весь вільний простір контейнера розподіляється у кожному
елементі додаючись до ширині контенту, тобто

       flex: 1 + 2 = 3  => 200 / 3 = 66.6px
       ширина контенту item1 = (100 +  66.6) ->  166.6px;
       ширина контенту item2 = (200px + 133.3px) ->  333.3px;

===========================================================
2. Встановити

 .item { flex-basis: 0 }
 .item1 { flex-grow: 1; }
 .item2  { flex-grow: 2;  }

Тепер розподіл вільного простору в контейнері браузер
здійснює  не враховуватиме ширину контенту !!! ,
і вільний простір розподіляється не між контентом, а між елементами

     padding + border = 50px + 50px = 100px
     remaining spacing = 600 - 100 = 500px
     flex = 1 + 2 = 3 => 500 / 3 = 166.6px

     ширина блоку item1 = 166.6px +  50px = 216.6;
     ширина блоку item2 = 333.3px  + 50px = 383,3;

============================================================
3.   Встановити  
 .item { flex-basis: 150px; }
 .item1 { flex-grow: 1; }
 .item2  { flex-grow: 2;  }

Загальна ширина віставлена за допомогою flex-basis = 2 * 150px = 300px
    flex = 1 + 2 = 3
    600 - 300px = 300 / 3 = 100px

ширина блоку item1 = 150px + 100px = 250px;
ширина блоку item1 = 150px + 200px = 350px;

============================================================
4. Встановити  
   .item  {flex-basis: 0;}
   .item1  {flex-basis: 150px; flex-grow: 1;}
   .item2 {flex-grow: 2;}

Загальна ширина flex-basis = 150px  + 50 = 200px
               flex = 1 + 2 = 3
               600 - 200px = 400 / 3 = 133px

ширина блоку item1 = 150px +  133px = 283px;
ширина блоку item2 =  133 * 2  + 50px  = 316px;
