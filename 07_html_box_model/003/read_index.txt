1. Для box__item встановити ширину 25%, padding: 1em;

2. Нам потрібно, щоб блоки розташовувалися поруч один з одним,
   і в той же час вони залишалися блоками 
   (тобто в них можна було мати блоковий контент)
   Для цього для box__item ставимо display: inline-block;

   При цьому незважаючи на те, що ми задали кожному блоку ширину
   25%, і в одному рядку має розташовуватися 4 блоки,
    у нас вийшло 3 блоки в рядку та пробіли між ними
    
    Це тому, що між блоками є перенесення рядка
    (згадуємо з одного заняття, що є пробілом в HTML коді)

3. Одним із способів видалення пробілів між елементами
    з свойством inline-block є наступний

   - контейнеру .box присвоїти font-size: 0;
   - елементу з класом .box__item присвоїти font-size: 18px;

4. Для box__item задати border 2px solid white;

5. Вирівнюємо текст по горизонталі центром блоку
   Для цього для елемента із класом .box__item задати 
   text-align: center;

6. Блоку Box 3 присвоїти width: 50%;
   При цьому ширина решти всіх блоків не повинна змінитися.
